## 四、问题解决（15%）

### 错误1：尝试修改元组元素

**错误现象：**
```python
versions = (3.8, 3.9, 3.10)
versions[0] = 3.11
# TypeError: 'tuple' object does not support item assignment
```

**原因分析：** 元组（tuple）是不可变数据类型，一旦创建就不能修改其元素。这与列表（list）形成对比。

**解决方案：** 
- 若需要修改，可先将元组转换为列表，修改后再转回元组
- 或者直接使用列表代替元组
- 元组适用于不需要修改的固定数据（如版本号、坐标等）

### 错误2：尝试将非数字字符串转换为整数

**错误现象：**
```python
int("abc123")
# ValueError: invalid literal for int() with base 10: 'abc123'
```

**原因分析：** `int()` 函数要求字符串只包含数字字符（和可选的正负号），遇到字母字符则无法转换。

**解决方案：**
- 在转换前使用 `str.isdigit()` 检查字符串内容
- 使用 `try/except` 捕获异常并给用户友好的错误提示
- 对于可能含字母的字符串（如十六进制），使用 `int(s, 16)` 指定基数
